# Simple Calculator

## Overview
This repository contains a simple calculator application implemented in Python. It is designed to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Improved handling of decimal points
- Enhanced error handling for division by zero

## Requirements
- Python 3.x

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/muskan-27-bot/Simple-calc.git
   ```
2. Navigate into the directory:
   ```bash
   cd Simple-calc
   ```
3. Install any required dependencies (if applicable).

## Usage
Simply run the calculator script and follow the on-screen prompts to perform calculations.

### Example:
```python
result = add(5, 3)
print(result)  # Outputs: 8
```

## Error Handling
- The calculator provides feedback for invalid input.
- Division by zero raises an informative error message to avoid unexpected application crashes.
